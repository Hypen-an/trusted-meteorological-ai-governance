# 前端使用说明

打开 `index.html` 即可使用, 不需要启动服务器。

## 用户测试建议

请受访者按以下顺序操作:

1. 查看“总览”, 了解四个通道的误差和风险等级。
2. 查看“黑盒治理”, 理解模型为什么不能只看预测图。
3. 查看“安全影响”, 理解错误预测和人身安全之间的关系。
4. 进入“风险计算”, 设置:
   - 通道: CH13 红外窗口
   - 目标区最低亮温: 180 K
   - 使用场景: 预警研判
5. 观察系统给出的风险等级和治理动作。
6. 回答问题: 这个结果能否直接用于灾害预警? 为什么?
7. 进入你单独制作的在线问卷, 填写量表和开放题。
8. 问卷题目可参考 `../03_问卷数据与反馈分析/问卷题目_气象AI预测系统信任与风险认知调查.md`。

## 论文中可使用的反馈指标

建议统计以下均值:

- 模型透明性
- 黑盒不确定感降低程度
- 风险提示有效性
- 人身安全影响理解程度
- 人工复核必要性认可度
- 适度信任形成程度

开放题可以用于总结系统优点和后续改进方向。
