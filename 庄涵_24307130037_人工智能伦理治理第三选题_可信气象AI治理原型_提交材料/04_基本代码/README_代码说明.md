# 基本代码说明

本文件夹仅保留与最终提交直接相关的基础代码。

## 保留代码

### `risk_scoring_reference.py`

用途:

- 说明前端风险计算器使用的规则驱动风险评分方法。
- 给出通道、亮温、云区结构、使用场景、训练分布外和数据质量问题的加权逻辑。
- 输出风险分数、风险等级和建议性治理动作。
- 不包含 PDF 生成流程, 不依赖模型权重或卫星原始数据。

运行方式:

```powershell
python risk_scoring_reference.py
```

注意:

- 该脚本只用于说明系统核心治理逻辑。
- 前端 `app.js` 中实现了同类风险计算逻辑。
